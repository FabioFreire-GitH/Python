# 📑 Estudos de Desenvolvimento e Programação Orientada a Objetos em Python

Este repositório foi criado para documentar a minha evolução na linguagem Python, consolidando conceitos fundamentais de lógica até a estruturação de códigos complexos e arquitetura de software.

O foco principal desta etapa de estudos é o domínio do paradigma orientado a objetos, essencial para a criação de sistemas Full-Stack robustos, escaláveis e de fácil manutenção.

## 🚀 Conceitos e Pilares Explorados:
- **Fundamentos Avançados:** Manipulação de estruturas de dados nativas (listas, dicionários, tuplas) e tratamento de exceções.
- **Abstração e Encapsulamento:** Criação de classes e objetos para modelar entidades do mundo real e proteger dados estruturais.
- **Herança e Polimorfismo:** Reutilização de código e criação de interfaces flexíveis para comportamentos dinâmicos.
- **Persistência de Dados (Opcional):** Integração básica com manipulação de arquivos ou conceitos iniciais de banco de dados.

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- **Ambientes Virtuais (venv)**
- **Paradigma de Programação Orientada a Objetos (POO)**
